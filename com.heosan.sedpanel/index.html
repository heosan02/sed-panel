Welcome to SED Panel
Scene Edit Detection panel for Adobe After Effects.
Automatic scene cut detection, quick navigation, and easy scene export.
Language
This guide appears once. Reopen anytime via About → Tutorial.
▶
DETECT SCENES
Select a video layer in the AE Timeline, then click DETECT SCENES.
💡 After Effects will automatically create markers at every detected cut point on the layer.
📌
Read Markers
Already have markers from AE manual detection?
Click Read Markers to read existing markers.
💡 Layer → Scene Edit Detection → Create Layer Markers → OK, then click Read Markers.
🖱️
Scene Navigation
Click a scene card → select and move CTI to that scene.
Ctrl+Click or Double-click → mark for export.
Thumbnails appear after clicking 🖼 Thumbs.
💡 Use ⏮ ◀ ▶ ⏭ buttons at the top for quick navigation between scenes.
✂
Cut Layer
Cut All Cut Points → split layer at all cut points (1 layer per scene).
Cut Selected Only → split only at marked scene cut points.
⚠ After splitting, layers cannot be automatically restored — but Ctrl+Z works.
🗑
Delete Except Selected
Mark scenes you want to keep, then click Delete All Except Selected.
⚠ This operation cannot be undone! Make sure marked scenes are correct before executing.
🎬
Export to Render Queue
Mark scenes to render, then click Export → Render Queue.
Each scene becomes a separate composition in AE Render Queue.
💡 After export, open Window → Render Queue, set output path, then Render All.
🖼
Thumbnail Preview
Click 🖼 Thumbs to generate thumbnails for each scene.
Fast mode (default) uses Python cv2 — generates all at once with a progress popup.
Lazy mode uses AE's saveFrameToPng — thumbnails appear one-by-one without a popup.
Switch modes in Settings → Thumbnail Mode.
💡 Thumbnails are rendered to the temp folder, then automatically cleaned after reading. Lazy mode is best for quick previews; Fast mode is best for speed with large scene counts.
⛓
Merge Scene
Mark 2 or more adjacent scenes, then click ⛓ Gabungkan Scene to combine them into one scene and one marker.
⚠ Only adjacent scenes can be merged (e.g. 1+2 or 5+6+7). Non-adjacent picks (e.g. 1+5) are skipped automatically, and if nothing is adjacent a warning will appear.
✅
